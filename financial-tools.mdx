## Usage Examples

### Stock Analysis Workflow
```python
# Get current stock data
price_data = call_tool("getPriceSnapshot", {"ticker": "AAPL"})

# Get historical performance
historical = call_tool("getHistoricalPrices", {
  "ticker": "AAPL", 
  "period": "1y",
  "interval": "1d"
})

# Get financial statements
financials = call_tool("getFinancials", {
  "ticker": "AAPL",
  "period": "quarterly",
  "limit": 8
})

# Calculate financial ratios
ratios = call_tool("getFinancialRatios", {
  "ticker": "AAPL",
  "period": "quarterly"
})
```
