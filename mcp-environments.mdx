## Overview

The **Theta Playground** is our environment that allows agents to complete tasks using our MCP tools in an isolated environment. We then evaluate the model's success in completing the task.

## Quick Start

### 1. Execute an Autonomous Task

Start streaming a accounting analysis task:

```bash
curl --no-buffer -X POST \
  -H 'Content-Type: application/json' \
  -H 'Accept: application/x-ndjson' \
  --data '{"task_id":"dimitri_holdings","api_key":"theta_XXXX","openai_api_key":"'"$OPENAI_API_KEY"'","model":"gpt-4o","max_steps":50}' \
  'https://playground.trythetasoftware.com/evaluate/stream?style=cli'
```
This will:
1. Create an isolated sandbox for the AI agent with the initial state of the task
2. Stream real-time actions as the model uses the tools to complete the task
3. Evaluate agent output accoring to a reward function to reveal a final score

**Available Task IDs:**
- `lbo-model-1` - LBO model completion with debt schedule calculations  
- `lbo-model-2` - Advanced LBO build with multi-layer debt and exit modeling  
- `finance-comps-jazz-1` - Pharmaceutical public comps build for Jazz/BMRN analysis  
- `re_model_ut` - Real estate property search and financial modeling
- `vc_darwin` - Venture capital funding structure analysis
- `bank-dividends` - Commercial bank DDM with CET1 constraints, projections, and terminal value
- `budget_analysis` - Ledger-based budget vs actual variance analysis using accounting tools
- `calendarization` - Complex quarterly financial data calendarization task
- `dimitri_holdings` - End-to-end small-business accounting: entries, trial balance, statements, close

## Evaluation
Evaluating the agent’s ability to complete the task is a critical component of the learning process. We assess the final state of the environment and assign a score based on how effectively the agent accomplished the task. For example, in the Real Estate Model task, we evaluate the accuracy of the agent’s inputs and return a score on a scale from 0.00 to 1.00, reflecting how closely the model matches the expected outcome.
