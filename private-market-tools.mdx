## Company Research Tools

### `getCrunchbaseCompanyInfo`
Get detailed company information from Crunchbase including name, description, funding, location, social networks, and more.

<Accordion title="View Parameters & Response">

**Parameters:**
```json title="getCrunchbaseCompanyInfo - Request"
{
  "company": "string (optional) - Company name or identifier (e.g., 'openai', 'google', 'microsoft'). Either company or domain must be provided.",
  "domain": "string (optional) - Company domain/website (e.g., 'openai.com', 'google.com', 'darwingov.com'). Either company or domain must be provided."
}
```

### `getCrunchbaseFundingRounds`
Get funding round information from Crunchbase with filtering options.

<Accordion title="View Parameters & Response">

**Parameters:**
```json title="getCrunchbaseFundingRounds - Request"
{
  "days_since_announcement": "integer (optional) - Number of days since funding announcement (1-360 days), default: 30",
  "investment_type": "string (optional) - Type of investment. Options: pre_seed, seed, series_a, series_b, series_c, etc.",
  "investor_identifiers": "string (optional) - UUID of specific investor to filter by"
}
```

**Response:**

  "success": true,

## Usage Examples

### Company Research Workflow
```python
# Research a company by name
company_info = call_tool("getCrunchbaseCompanyInfo", {
  "company": "openai"
})

# Research a company by domain
company_by_domain = call_tool("getCrunchbaseCompanyInfo", {
  "domain": "darwingov.com"
})

# Find recent seed funding rounds
recent_seed_rounds = call_tool("getCrunchbaseFundingRounds", {
  "investment_type": "seed",
  "days_since_announcement": 30
})

# Find all Series A rounds in the last 60 days
series_a_rounds = call_tool("getCrunchbaseFundingRounds", {
  "investment_type": "series_a",
  "days_since_announcement": 60
})
```

### Market Intelligence Analysis
```python
# Analyze funding trends
recent_rounds = call_tool("getCrunchbaseFundingRounds", {
  "days_since_announcement": 90
})

# Research specific companies in a sector
ai_companies = [
  call_tool("getCrunchbaseCompanyInfo", {"company": "openai"}),
  call_tool("getCrunchbaseCompanyInfo", {"company": "anthropic"}),
  call_tool("getCrunchbaseCompanyInfo", {"company": "cohere"})
]
```
