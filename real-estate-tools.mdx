## Usage Examples

### Investment Property Analysis
```python
# Find properties in target market
properties = call_tool("getPropertyRecords", {
  "city": "Austin",
  "state": "TX",
  "bedrooms": 3,
  "priceMax": 500000
})

# Get value and rent estimates
for property in properties:
  value_estimate = call_tool("getValueEstimate", {
    "address": property["address"]
  })
  
  rent_estimate = call_tool("getRentEstimate", {
    "address": property["address"]
  })

# Compare multiple markets
market_comparison = call_tool("compareMarkets", {
  "zipCodes": ["78701", "78702", "78703"]
})
```

### Construction Cost Analysis
```python
# Get material costs
concrete_costs = call_tool("searchSources", {
  "search_term": "concrete",
  "state": "Texas",
  "county": "Travis"
})

# Get labor rates
labor_rates = call_tool("getLaborRates", {
  "trade": "contractor",
  "state": "Texas",
  "county": "Travis"
})

# Get equipment rental costs
equipment_costs = call_tool("getEquipmentRentalRates", {
  "equipment_type": "excavator",
  "state": "Texas", 
  "county": "Travis"
})
```
