## Account Balance Tools

### `ledger_balance`
Shows account balances with powerful filtering options for comprehensive financial reporting.

<Accordion title="View Parameters & Response">

**Parameters:**
```json title="ledger_balance - Request"
{
  "query": "string (optional) - Query pattern to filter accounts (e.g., 'Assets', 'Expenses:Food')",
  "date_range": "string (optional) - Date range for the report (e.g., '2024', 'last month', 'Q1 2024')",
  "display_options": "object (optional) - Additional display formatting options"
}
```

**Response:**
```json title="ledger_balance - Response"
{
  "success": true,
  "balance_report": "Formatted balance report showing account totals",
  "total_balance": "Summary of filtered accounts",
  "query_applied": "Assets",
  "date_range": "2024"
}
```

</Accordion>

---

### `ledger_register`
Shows transaction register with detailed history for account analysis.

<Accordion title="View Parameters & Response">

**Parameters:**
```json title="ledger_register - Request"
{

  "date_range": "string (optional) - Date range for transactions",

```

**Response:**
```json title="ledger_register - Response"

  "register_report": "Detailed transaction register with dates, descriptions, and amounts",
  "transaction_count": 45,
  "date_range": "last month",
  "total_amount": "$2,450.00"
}
```

</Accordion>

</Accordion>

</Accordion>

## Usage Examples

### Personal Finance Analysis
```python
# Check account balances
balances = call_tool("ledger_balance", {
  "query": "Assets",
  "date_range": "2024"
})

# Review spending patterns
food_expenses = call_tool("ledger_register", {
  "query": "Expenses:Food",
  "date_range": "last 3 months"
})

# Get budget variance report
budget_analysis = call_tool("ledger_budget", {
  "query": "Expenses",
  "date_range": "2024",
  "reporting_period": "monthly"
})
```

### Business Accounting Workflow
```python
# Generate income statement data
revenue_accounts = call_tool("ledger_balance", {
  "query": "Income",
  "date_range": "2024"
})

expense_accounts = call_tool("ledger_balance", {
  "query": "Expenses", 
  "date_range": "2024"
})

# Analyze cash flow
cash_flow = call_tool("ledger_register", {
  "query": "Assets:Cash",
  "date_range": "Q4 2024"
})

# Get detailed transaction data
transactions = call_tool("ledger_print", {
  "query": "tag:business",
  "date_range": "2024"
})
```

### Investment Portfolio Tracking
```python
# Track investment accounts
investment_balances = call_tool("ledger_balance", {
  "query": "Assets:Investments",
  "date_range": "2024"
})

# Review investment transactions
investment_history = call_tool("ledger_register", {
  "query": "Assets:Investments:Stocks",
  "date_range": "last year"
})

# Check portfolio commodities
portfolio_assets = call_tool("ledger_commodities", {
  "query": "AAPL|MSFT|GOOGL"
})
```

### Advanced Reporting
```python
# Custom balance sheet depth
balance_sheet = call_tool("ledger_raw_command", {
  "command_args": ["bal", "--depth", "3", "Assets", "Liabilities"]
})

# Profit and loss statement
pnl = call_tool("ledger_raw_command", {
  "command_args": ["bal", "--period", "monthly", "Income", "Expenses"]
})

# Account statistics
stats = call_tool("ledger_stats", {
  "query": "2024"
})
```

## Key Features

- **Double-Entry Accounting**: Full support for professional double-entry bookkeeping
- **Flexible Reporting**: Generate balance sheets, income statements, and custom reports
- **Budget Analysis**: Track budget vs actual with variance reporting
- **Multi-Currency Support**: Handle multiple currencies and commodities
- **Powerful Querying**: Advanced filtering and searching capabilities
- **Raw CLI Access**: Direct access to Ledger CLI for advanced operations
- **Transaction Analysis**: Detailed transaction history and register views
